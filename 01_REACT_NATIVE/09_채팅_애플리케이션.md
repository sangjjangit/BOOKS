# 09_채팅_애플리케이션


## 9.1 프로젝트 준비

#### - 내비게이션
- npm install #react-navigation/native
- npm install #react-navigation/stack #react-navigation/bottom-tabs

#### - 라이브러리
- npm install styled-components prop-types

- expo-image-picker: https://bit.ly/expo-imagepicker
: 기기의 사진이나 영상을 가져올 수 있도록 시스템 UI 에 접근할 수 있는 기능 제공
: npm install expo-image-picker

- moment: https://momentjs.com
: 시간과 관련된 많은 기능 제공
: npm install moment

- react-native-keyboard-aware-scroll-view: https://bit.ly/keyboard-scroll
: 키보드가 화면을 가리면서 생기는 불편한 점 해결을 위한 라이브러리
: npm install react-native-keyboard-aware-scroll-view

- react-native-gifted-chat: https://bit.ly/gifted-chat
: 메시지를 주고받는 채팅 화면을 쉽게 구현하기 위한 라이브러리
: npm install react-native-gifted-chat


## 9.2 파이어베이스
- 인증, 데이터베이스 등의 다양한 기능을 제공하는 개발 플랫폼
- 파이어베이스 콘솔: https://console.firebase.google.com/
- 인증: 이메일/비밀번호
- 데이터베이스: 파이어스토어, 실시간 데이터베이스 두 가지 종류 제공
- 스토리지: 파이어베이스 유료 계정으로 변경, 무료 초과시 결제된다.
- 라이브러리 설치: `npx expo install firebase` or `npm install firebase`
