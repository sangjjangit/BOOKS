## 2.2 리액트 네이티브 프로젝트 만들기
- expo 이용하는 방법: 처음 접하는 분 추천
- 리액트 네이티브 CLI 를 이용하는 방법: 처음부터 네이티브 모듈을 직접 건드려야 한다면 리액티 네이티브 CLI 로 시작

### expo
- node version 17 이하
npm install --global expo-cli
expo init my-app
cd my-app
npm start

- node version 17 이상
npx create-expo-app my-app
cd my-app
npx expo start

### 리액트 네이티브 CLI
npx react-native init my-app
cd my-app
npm run ios | npx react-native run-android
npm run android | npx react-native run-ios
